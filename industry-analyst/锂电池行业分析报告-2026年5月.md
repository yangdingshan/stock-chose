## 一、行业定义与范围

### 行业边界

本次分析的行业为**锂离子电池行业**（申万三级分类：锂电池），包括动力电池、储能电池、消费电子电池三大应用方向。

- **一级行业**：电力设备
- **二级行业**：电池
- **三级行业**：锂电池（电芯制造及上游材料与设备）

### 产业链结构

```
上游资源（矿/盐湖）         中游材料             中游电芯/设备          下游应用
─────────────────────────────────────────────────────────────────────────────
锂矿（天齐/赣锋/盐湖）  →  正极材料（成本占比40%）→  动力电池（65-70%）→ 新能源乘用车
钴矿（华友/洛阳钼业）   →  负极材料（成本占比10%）→  储能电池（30%+） → 储能电站/数据中心
镍矿（华友/格林美）     →  电解液（成本占比10%）  →  消费电池        → 手机/两轮车
锰矿                   →  隔膜（成本占比15%）   →  锂电设备         → 低空经济/机器人
                        →  铜箔/铝箔/结构件    →                    → 电动重卡/商用车
                        →  碳酸锂/六氟磷酸锂
                        →  PVDF/导电剂/碳纳米管
```

**本次分析覆盖**：上游锂资源 → 中游四大主材 → 电芯制造 → 锂电设备，以中游电芯制造和材料环节为分析重点。

## 二、产业生命周期判断

### 总体判断：锂电池行业处于**成长期后段向成熟期过渡**

### 核心依据——渗透率数据

| 细分市场 | 当前渗透率 | 所处阶段 |
|----------|-----------|---------|
| 中国新能源乘用车 | **约55%**（2025年达53.9%） | 成熟期前段 |
| 国内电动重卡/商用车 | **15%-29%** | 成长期 |
| 欧洲新能源车 | 约30% | 成长期 |
| 全球储能（电化学） | **15%-25%** | 成长期前段 |
| 全球新能源车整体 | 约25-30% | 成长期 |
| 新兴市场（东南亚/南美/非洲） | <10% | 导入期 |

### 分应用场景的生命周期矩阵

| 应用场景 | 生命周期阶段 | 渗透率 | 投资关键词 |
|----------|-------------|--------|-----------|
| **中国新能源乘用车** | 成熟期前段 | ~55% | 增速放缓、格局优化、龙头溢价 |
| **电动重卡/商用车** | 成长期 | 15-29% | 高增长、带电量大（7倍乘用车） |
| **全球储能** | 成长期前段 | 15-25% | **本轮周期最大增量**，CAGR 50%+ |
| **低空经济/飞行电池** | 导入期 | <5% | 远期期权，高能量密度需求 |
| **人形机器人** | 导入期 | <5% | 远期期权 |
| **消费电子** | 成熟期 | >80% | 稳定市场 |
| **两轮车/电动工具** | 成熟期前段 | ~50% | 钠电替代机会 |

### 特殊说明：第三轮景气上行周期已确认

锂电池行业已完成三轮完整周期，当前正处于**第三轮上行周期的繁荣阶段**：

| 周期 | 时间 | 核心驱动 | 阶段特征 |
|------|------|----------|---------|
| 第一轮 | 2015-2018 | 补贴刺激、特斯拉Model 3 | 导入期→成长期过渡 |
| 第二轮 | 2020-2022 | 碳中和、欧洲碳排放、产品力跃升 | 成长期爆发 |
| **第三轮** | **2025开启** | **储能商业模式成熟、AI数据中心缺电、光储平价** | **由储能驱动的供需紧平衡** |

**关键判断**：行业并非线性演进，而是呈现"**周期化的成长期**"特征——长期趋势向上，但伴随3-4年一轮的产能周期波动。本轮周期不同于前两轮，驱动引擎已从"**新能源车**"切换到"**储能**"，增长更具持续性。

## 五、卡脖子环节深度分析

### 卡脖子全景图

锂电池行业经历二十余年国产替代，"卡脖子"环节已大幅减少，但仍在**锂资源、高端设备、特种材料**三大领域存在不同程度的依赖。

```

  ↑

95% │ 电解液、负极材料 ★★★★★    （全球主导）

```

### 卡脖子环节一：锂资源——对外依赖度约70%（最严重的"卡脖子"）

**现状**：
- 中国锂电池产量占全球**70-80%**，但锂资源约**70%依赖进口**
- 2025年进口锂精矿约**775.1万吨**（同比+39.4%），主要来自澳大利亚、津巴布韦、巴西
- 净进口碳酸锂**23.7万吨**（同比+2.8%）
- 海外资源国政策收紧：津巴布韦2026年2月全面暂停锂精矿及原矿出口、智利/墨西哥将锂列为战略资源

**重大转折——"亚洲锂腰带"**：
- 2025年9月发现横跨四川、青海、西藏、新疆的**2800公里"亚洲锂腰带"**
- 中国锂资源储量全球占比从**6%跃升至16.5%**，排名从全球第六升至**第二**
- 盐湖提锂技术突破：成本压低至**2.4万元/吨**（硬岩提锂成本6-8万/吨）

### 卡脖子环节三：铜箔核心设备——阴极辊（已基本突破）

## 九、综合结论与风险提示

### 核心结论

1. **锂电池行业处于成长期后段向成熟期过渡，2025年确认进入第三轮景气上行周期**。本轮周期的核心驱动力已从新能源车切换到储能，增长的持续性和质量更高。

2. **"卡脖子"现状总结**：
   - **最严重**：锂资源对外依赖度约70%（但随着"亚洲锂腰带"发现+盐湖提锂突破，2026年有望逆转）
   - **中等**：高端导电炭黑（<20%）、高精度涂布模头（<30%）
   - **已突破**：隔膜设备（>80%）、铜箔阴极辊（>45%）、PVDF粘结剂（>87%）、碳纳米管（>70%）
   - **全球领先**：涂布机/卷绕机/叠片机（>95%）、电解液/负极/磷酸铁锂（全球绝对主导）

3. **投资主线排序**：
   - **确定性最强**：宁德时代（极宽护城河+储能爆发+出口加速+PE 19-22x）
   - **卡脖子弹性**：天齐锂业/赣锋锂业（锂资源）+ 黑猫股份（导电炭黑）+ 璞泰来（PVDF+隔膜设备）
   - **弹性最大**：天赐材料/多氟多（电解液/6F涨价）
   - **性价比最高**：湖南裕能（LFP龙头）、尚太科技（PE~11x）、恩捷股份（隔膜壁垒+供需趋紧）
   - **设备国产替代**：先导智能（固态整线）、泰金新能（阴极辊）、洪田股份（铜箔设备）

4. **2026年是利润兑现年**：2025Q1行业利润同比+58%，2026Q1同比+88%，Q2预期超越2021-2022年单季高点。多环节PE已降至11-15x，PEG<1，具备戴维斯双击条件。

### 风险提示

| 风险 | 影响程度 | 触发条件 |
|------|---------|---------|
| **碳酸锂价格大幅回落** | 高 | 矿山集中复产+需求低于预期，价格跌破10万元/吨 |
| **储能需求不及预期** | 高 | 国内容量电价政策执行不力、海外AI资本开支缩减 |
| **钠电池加速替代** | 中 | 锂价持续高位+钠电成本降至0.3元/Wh以下 |
| **欧美贸易壁垒升级** | 中 | 关税大幅提高或禁止中国电池进口 |
| **技术路线颠覆** | 中 | 全固态电池量产速度超预期 |
| **资源国政策风险** | 中 | 津巴布韦/智利等国进一步收紧锂矿出口 |
| **车企自建电池产能** | 低中 | 小米、蔚来等车企自建电池产线 |

### 可证伪条件

- 如果2026H2碳酸锂价格跌破10万元/吨 → "供需紧平衡"判断错误
- 如果宁德时代2026年单位盈利跌破0.07元/Wh → 竞争格局恶化
- 如果2026年全球储能新增装机增速低于30% → 储能爆发逻辑需修正
- 如果钠电池2027年成本降至0.3元/Wh且大规模量产 → 锂电池天花板下移
- 如果中国锂资源自给率2027年仍低于40% → "亚洲锂腰带"贡献被高估
