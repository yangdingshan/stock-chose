# 港股通高股息标的筛选分析报告

> 报告日期：2026年6月4日
> 筛选标准：股息率 ≥ 4%，PE ≤ 15倍（优先），港股通可交易标的
> 数据来源：港交所、东方财富、中银国际、经济通、智通财经等

---

## 一、高股息投资的核心逻辑

### 1.1 港股高股息的独特优势

| 维度 | 港股高股息 | A股高股息 |
|------|-----------|----------|
| 平均股息率 | 5-8%（内银/能源/电信） | 3-5% |
| 估值水平 | PE 4-8倍（央国企） | PE 6-12倍 |
| 分红文化 | 国际机构股东推动，分红率稳定 | 近年才逐步改善 |
| 可投标的 | 汇丰、中移动等国际化企业 | 以国内央企为主 |
| 汇率敞口 | 港币挂钩美元，对冲人民币贬值 | 纯人民币资产 |

### 1.2 2026年宏观背景

- **全球利率下行周期**：主要央行逐步降息，高股息资产的相对吸引力上升
- **中国无风险利率下行**：10年期国债收益率约1.7-1.8%，股息率优势更加突出
- **南向资金偏好**：银行板块年内净买入超1,435亿港元，高股息是核心配置逻辑
- **央企市值管理**：国资委推动央企加强股东回报，分红率和回购力度持续提升

### 1.3 港股通股息税提醒

> ⚠️ **重要**：通过港股通投资H股，股息红利所得税为**20%**。例如某H股派息率6%，实际到手约4.8%。红筹股视注册地而定，部分可能更高。本地港股的股息税率因公司注册地不同而有差异。

- 20%股息税后实际收益打八折

## 八、风险提示

1. **股息税损耗**：港股通投资者持有H股需缴纳20%红利税，实际到手收益=（1-20%）× 名义股息率
2. **分红可持续性**：高股息不等于可持续，需关注公司自由现金流和分红率是否过高
3. **价值陷阱风险**：股价下跌可能使股息率"被动升高"，需区分真高股息与基本面恶化
4. **行业周期风险**：能源、煤炭受商品价格影响；银行受经济周期和利率环境影响
5. **汇率风险**：港币与美元挂钩，人民币升值会侵蚀港币计价的分红收益
6. **流动性风险**：部分中小盘高息股流动性差，大额进出冲击成本高
7. **老千股风险**：警惕频繁供股/合股、业绩亏损但不退市、关联交易异常的高息"陷阱"

## 附录：股息率计算公式

```
名义股息率 = 每股年度分红（含特别息） / 当前股价 × 100%
实际到手股息率 = 名义股息率 × (1 - 适用税率)
```

**港股通股息税标准**：
- H股（注册地在内地的香港上市公司）：**20%**
- 红筹股（注册地在海外、控制权在内地的公司）：**10%-20%**（视注册地）
- 香港本地股：视公司注册地，部分**0%或10%**
