## 一、行业定义与范围

### 行业边界

本次分析的行业为**贵金属行业**，涵盖黄金、白银、铂族金属（铂、钯、铑、铱、锇、钌）的矿产资源开采、冶炼加工、回收再利用及下游新材料应用。

- **一级行业**：有色金属
- **二级行业**：贵金属 / 工业金属
- **三级行业**：黄金采选冶炼、白银冶炼、铂族金属回收与新材料

### 产业链结构

```
上游（矿产资源）              中游（冶炼加工）                下游（应用与消费）
─────────────────────────────────────────────────────────────────────────────
金矿开采（紫金/山金/中金）→ 黄金冶炼/精炼（纯度99.99%）  → 金饰消费（占比~45%）
银矿/伴生银回收            → 白银电解精炼                 → 央行购金（占比~20%）
铂族金属矿（南非/俄罗斯）★  → PGM分离提纯（技术壁垒极高） → 黄金ETF/投资金条（~25%）
贵金属二次资源回收 ★★       → 高纯靶材/浆料/催化剂制备 ★★ → 工业应用（电子/汽车/化工~10%）
                                                           → 氢能（PEM电解水/燃料电池）
                                                           → 半导体（溅射靶材）
```

**本次分析覆盖**：黄金采选冶炼全链条、铂族金属回收与新材料为卡脖子重点，兼顾白银龙头。

### 贵金属的特殊性

与一般工业品不同，贵金属具有**三重属性**：

| 属性 | 黄金 | 白银 | 铂族金属 |
|------|------|------|---------|
| **商品属性** | 金饰、工业 | 工业（光伏/电子~50%）、银饰 | 汽车催化剂、化工、氢能 |
| **货币属性** | 央行储备、对冲法币信用 | 历史货币、工业消耗 | 铂金币（次要） |
| **金融属性** | 利率/美元/避险定价 | 跟随黄金+工业需求 | 工业周期主导 |

**关键特征**：黄金是"利率的镜像"——实际利率越低（甚至为负），黄金的持有机会成本越低，价格越高。2024-2026年全球央行持续购金是本轮金价牛市的底层驱动力之一。

## 二、产业生命周期判断

### 总体判断：黄金处于**成熟期（稳定扩张+周期化特征）**，铂族金属处于**成长期（氢能+新材料驱动）**

**核心依据——渗透率与供需格局**：

| 维度 | 黄金 | 白银 | 铂族金属 |
|------|------|------|---------|
| **已探明储量可采年限** | ~15-18年 | ~20年 | ~30年+（南非资源丰富） |
| **全球矿山产量趋势** | 年增~1-2%，近峰值 | 年增~1%，伴生矿为主 | 年增~3%，南非/俄罗斯主导 |
| **回收占供给比例** | ~25-30% | ~20% | ~25-35%（中国<5%★） |
| **需求增长驱动力** | 央行购金+投资避险 | 光伏银浆+工业 | 氢能催化剂+汽车催化剂 |
| **对应生命周期阶段** | **成熟期** | **成熟期（工业属性增强）** | **成长期前段（新需求）** |

### 黄金：典型成熟期特征，但存在"第二增长曲线"催化

黄金采选冶炼是典型的成熟期行业，但本轮周期存在显著差异：

1. **全球央行持续净购金**：2022年1,082吨 → 2023年1,037吨 → 2024年1,045吨 → 2025年约1,100吨+，连续4年超千吨，重塑了黄金的需求结构
2. **去美元化浪潮**：金砖国家、中东主权基金增配黄金，替代美债储备
3. **地缘政治常态化**：黄金作为"终极避险资产"的配置价值持续提升
4. **"央行购金"等同于开辟了黄金的第二增长曲线**，使传统基于实际利率-ETF持仓的定价框架阶段性失效

### 3A. 防守性分析——护城河

#### 黄金采矿：资源垄断型护城河

黄金矿山的核心壁垒是**矿产资源**（稀缺性+品位+储量寿命）：

| 护城河类型 | 具体表现 | 中国代表企业 |
|-----------|---------|------------|
| **资源垄断** | 优质金矿资源稀缺，新发现大型金矿极少 | 紫金矿业（全球布局）、山东黄金（胶东半岛） |
| **规模经济** | 大规模露天开采成本远低于小矿山 | 紫金矿业（全球前10大金矿企业） |
| **技术壁垒** | 难处理金矿（卡林型、高砷高硫）选冶技术 | 紫金矿业（生物氧化/高压预氧化） |
| **产业链整合** | 采矿→选矿→冶炼→精炼→深加工一体化 | 紫金矿业、山东黄金、中金黄金 |

#### 产业链利润分配

- **采矿端**（紫金/山金/中金）：享受金价上涨的全部弹性，利润最厚

- **珠宝零售端**（周大福/老凤祥）：品牌溢价+渠道，与金价波动弱相关
- **回收端**（贵研铂业★）：技术壁垒高，铂族金属回收毛利率>15%

### 估值逻辑框架

**黄金股适用"周期股估值"框架**——在景气度底部买入（高PE/亏损），在景气度顶部卖出（低PE/暴利）：

| 估值方式 | 适用场景 | 当前状态 |
|---------|---------|---------|
| **PB估值** | 金价低迷期，利润被压缩时 | PB 2-5x区间，历史中位数偏上 |
| **低PE买入** | 金价上行期，利润快速释放 | PE 14-44x，分化明显，紫金/赤峰PE偏低 |
| **NAV/资源价值** | 评估矿产资源内在价值 | 紫金矿业NAV折价约15-20% |
| **PE×金价弹性** | 预测景气周期利润区间 | 金价每+10%，紫金利润弹性约+15-18% |

### 5.1 贵金属行业"卡脖子"全景图

```
卡脖子严重程度：

★★★★★ 铂族金属（PGM）：中国自给率 < 2%，98%+依赖进口 → 氢能战略的"阿喀琉斯之踵"
★★★★  铑/铱/钌：伴生于铂矿，南非/俄罗斯绝对垄断
★★★   高纯贵金属靶材/电子浆料：半导体+光伏卡点
```

**进口来源格局**：

| 品种 | 主要进口来源 | 进口依赖度 | 地缘风险 |
|------|------------|----------|---------|
| 铂 | 南非（~70%）、俄罗斯（~15%） | **>97%** | 高（俄制裁+南非电力危机） |

**卡脖子根因**：
1. 中国铂族金属资源极度贫乏，已探明储量仅占全球~0.5%
2. 铂族金属是氢能产业（PEM电解水、燃料电池）的关键催化剂材料
3. 2025年全球氢能产业消耗铂约12吨，预计2030年增至50-70吨
4. 如氢能战略大力推进，铂族金属供需缺口将进一步扩大

**国产替代进展**：

| 燃料电池膜电极 | ~25% | 贵研铂业/亿华通 | ★★★★★ |
| 汽车尾气催化剂 | ~35% | 贵研铂业 | ★★★ |
| 铂族金属靶材 | ~10% | 贵研铂业/有研新材 | ★★★★★ |

#### 卡脖子二：高纯贵金属靶材与电子浆料 ★★★

**现状**：
- 半导体用高纯金靶材（99.999%+）、银靶材国产化率不足15%

**卡脖子根因**：高纯度（5N-6N）、高致密度、微结构控制的粉末冶金技术壁垒极高

## 九、综合结论与风险提示

### 9.1 核心判断

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  贵金属（尤其是黄金）处于"超级周期"中段，不是终点               │
│                                                                 │
│  三重驱动力：                                                   │
│  ① 地缘政治常态化 → 避险需求持续                                │
│  ② 去美元化 + 央行购金 → 结构性需求增量（最大变量）            │
│  ③ 美国财政赤字恶化 → 美元信用下降                             │
│                                                                 │
│  核心投资结论：黄金股仍有配置价值，选股重于择时                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 主要风险

| 风险类型 | 风险描述 | 影响程度 | 应对策略 |
|---------|---------|---------|---------|
| **金价回调风险** | 若美国经济走强，美联储重回加息，金价可能回调至$3,500以下 | ★★★★★ | 控制仓位，分批建仓 |
| **俄乌停战风险** | 若地缘政治风险消退，避险需求减弱 | ★★★★ | 关注央行购金能否对冲 |
| **替代品风险** | 比特币现货ETF分流黄金投资需求 | ★★★ | 央行购金不买BTC，阶段性影响有限 |
| **成本上升风险** | 矿山品位下降+能源/人工成本上升，压缩利润 | ★★★ | 优选低成本矿山公司 |
| **海外政策风险** | 加拿大/澳大利亚收紧外资矿业并购审查 | ★★★ | 关注已在海外站稳的公司 |
| **氢能路线风险** | 若燃料电池低铂化/无铂化技术突破，PGM需求逻辑受损 | ★★★ | 跟踪AEM/碱性电解槽进展 |
| **流动性风险** | 小市值贵金属股流动性差，大资金进出困难 | ★★ | 以龙头为主，小票控制比例 |

### 9.3 可证伪条件

以下条件出现时，上述结论需要修正：

1. **全球央行连续2个季度净售金**（而非购金）→ 黄金结构性需求逻辑被证伪
2. **美国10年期TIPS收益率持续高于2.5%** → 黄金持有机会成本大幅上升
3. **美联储明确释放加息信号** → 实际利率预期逆转
4. **全球地缘政治全面缓和**（俄乌停战+中东和平+中美关系改善）→ 避险溢价消退
5. **铂族金属回收率在中国达到>50%** → 供给改善，需重新评估供需缺口
