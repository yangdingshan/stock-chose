## 一、港股通机制概述

### 1.1 什么是港股通

港股通是中国内地投资者通过上海证券交易所（沪港通）和深圳证券交易所（深港通）直接投资香港联交所上市股票的机制。内地投资者可使用人民币普通股票账户，通过境内券商直接买卖合资格港股。

### 1.2 当前规模

| 指标 | 数据 |
|------|------|
| 港股通标的总数 | **约550只**（沪港通+深港通合计，含重叠部分） |
| 沪港通港股标的 | 约320只 |
| 深港通港股标的 | 约500只 |
| 年内南向资金净流入 | **超2,800亿港元**（截至2026年6月2日） |
| 南向资金整体持股比例 | **19.61%**（截至2026年5月28日） |
| 南向持股超20%个股数 | **253只** |

### 1.3 纳入标准

港股通标的筛选主要依据恒生综合指数（HSCI）成分股：

- **恒生综合大型股指数**成分股 → 自动纳入
- **恒生综合中型股指数**成分股 → 自动纳入
- **恒生综合小型股指数**成分股 → 需满足日均市值>50亿港元及流动性要求
- **AH股**（A+H双重上市公司）→ A股上市后自动纳入H股部分
- **新上市证券** → 价格稳定期结束后评估纳入

**定期调整**：跟随恒生综指每半年检讨（通常3月和9月生效）
**临时调整**：新上市AH股、特殊纳入等

## 三、南向资金流向分析

### 3.1 2026年整体趋势

| 时间维度 | 净流入/流出 | 关键特征 |
|----------|------------|----------|
| 2026年1月-4月 | +2,700亿港元 | 持续大幅净买入 |
| 2026年5月 | -35.5亿港元 | 年内首次单月净流出 |
| 2026年6月初（1-2日） | +68.8亿港元 | 净流入恢复 |
| **年内累计** | **+2,800亿港元** | 远超历史同期 |

### 3.2 行业资金偏好（2026年前5个月）

南向资金呈现明显的**"杠铃策略"**：

| 板块 | 净买入金额 | 资金态度 | 逻辑 |
|------|-----------|---------|------|
| 🏦 **银行** | 约**1,435亿港元** | 🔥 最大重仓方向 | 高股息防御，PE仅4-5倍 |
| 💻 **硬件/半导体** | 约**978.5亿港元** | 🔥 硬科技加仓 | 业绩兑现确定性强 |
| ⚡ **石油石化** | 约**96亿港元**（5月） | 🔥 红利+地缘驱动 | 中海油等央企高分红 |
| 🌐 **互联网/软件** | 净卖出约**107亿港元**（5月） | ⬇️ 遭抛售 | 资本开支大、回报未显 |
| 🛒 **可选消费零售** | 净卖出约**169亿港元**（5月） | ⬇️ 分歧加大 | 竞争格局恶化 |

### 3.3 核心结论

> **资金路径已明确：从互联网软科技 → 半导体硬科技 + 能源/银行红利。**
> 
> 市场主线从"AI投入了多少"转向"AI赚不赚钱"，半导体硬件订单和业绩率先兑现，而互联网巨头资本开支巨大但回报周期长，阶段性遭减持。

```

│              2026年港股通杠铃配置                       │
```

### 5.2 行业配置建议

| 配置类型 | 行业 | 建议仓位 | 配置逻辑 |
|----------|------|---------|----------|
| **核心底仓** | 银行、电信 | 25-30% | 高股息防御，类债券属性，PE仅4-5倍 |
| **卫星成长** | 半导体、AI硬件 | 20-25% | 业绩兑现确定性强，国产替代+AI双驱动 |
| **红利增强** | 能源（石油/煤炭） | 10-15% | 高分红+地缘政治溢价，商品属性提供弹性 |
| **消费复苏** | 汽车、家电 | 10-15% | 以旧换新政策催化，出口竞争力提升 |
| **医药创新** | 生物科技 | 5-10% | 创新药出海加速，估值处于历史底部 |
| **现金/观望** | 互联网平台 | 0-10% | 等待AI商业化确认信号 |

### 5.3 风险提示

1. **汇率风险**：港股通投资以港币计价，人民币汇率波动直接影响投资回报
2. **流动性风险**：港股中小盘股票流动性远低于A股，大额进出可能产生显著冲击成本
3. **股息税差异**：通过港股通持有H股/红筹股，股息红利税为20%（高于A股持股1年以上的0%）
4. **无涨跌停限制**：港股单日波动可能远超A股，极端行情下可能单日跌超50%
5. **老千股风险**：需警惕频繁供股/合股、长期亏损不退市、关联交易频繁的"老千股"特征
6. **政策风险**：香港市场受中美关系、行业监管政策等多重外部因素影响
7. **地缘政治风险**：外资占比高的个股受全球资金流向和地缘政治事件影响显著
