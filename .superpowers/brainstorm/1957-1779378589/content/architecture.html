三层架构

架构总览
🖥 前端层 (Thymeleaf + Bootstrap 5)
仪表盘 / 全部排名 / 下载管理 / 指数详情 — 四个页面
↓ HTTP + AJAX
⚙ 服务层 (Spring Boot)
StockService (排名算法) · DataDownloadService (下载编排) · IndexService (指数处理)
↓
📡 数据源适配层 (DataSource Adapter)
EastmoneyAdapter · AKShareAdapter · TushareAdapter — 主备自动切换
↓
💾 持久层
MySQL (股票+排名数据) · 本地文件缓存 (下载快照备份)

数据下载流程（多数据源冗余）

容错下载策略
1. 优先尝试 东方财富 API (免费、数据全)
3次重试 + 指数退避 + 随机UA
2. 失败则切换 AKShare (开源Python库,通过HTTP调用)
启动本地Python微服务或直接HTTP代理
3. 都失败则加载 本地缓存快照
上次成功下载的数据存为 JSON 备份，保证系统始终可用
每次成功下载后自动更新本地缓存。前端显示数据来源和更新时间。
