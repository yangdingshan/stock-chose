### 6.1 PE估值

- 当前PE (TTM): 28.23x
- 5年历史PE: 最低15x / 中位35x / 最高80x (上市后高估值已大幅消化)
- 当前PE历史分位: 30% (处于历史中下区间)
- 行业平均PE (2026E): 22.5x
- 合理PE判断: **22-25x** (基于: 2026E净利28亿、增速>90%、行业Top 3地位)
- **合理估值: HK$32 - HK$36**

### 6.2 PEG估值

- 预测3年净利润CAGR (2025-2028E): ~70% (按花旗/中金预测均值)
- PEG = PE / CAGR = 28.23 / 70 = **0.40** (远低于1)
- PEG=1对应合理PE: 70x → 当前低估严重
- 按PEG=0.8-1.0 (合理成长溢价) 对应PE 56-70x → 估值上限HK$66-82
- 保守取PEG=0.5-0.6对应PE 35-42x
- **合理估值: HK$45 - HK$50**

### 6.3 PB估值

- 当前PB: 1.21x
- 行业平均PB: 2.8x (宁德4.2x、亿纬2.5x、比亚迪3.8x)
- 公司历史PB: 最低0.8x / 中位1.8x / 最高5.5x
- ROE (2025): 3.5% → 折现率8% → 理论合理PB = 3.5/8 = 0.44x (偏低，因ROE仍在快速提升期)
- 按2026E ROE 5.5%估算 → 合理PB = 5.5/8 = 0.69x (静态)
- 按2027E ROE 8%估算 → 合理PB = 8/8 = 1.0x (动态)
- 取折中考虑业绩拐点+ROE提升趋势
- **合理估值: HK$28 - HK$32**

## 九、核心逻辑与风险

### 9.1 核心增长逻辑

1. **行业层面 - 全球电动化+储能爆发:** 全球电动车渗透率2025年20% → 2028年30%+，储能装机量CAGR 35%+，锂电池总需求复合增速15-18%；行业集中度提升后Top 3享受规模红利
2. **公司层面 - 三大成长曲线:**
   - **储能业务** (高增长): 2025年+115% → 2026E +60% → 2027E +40%，单瓦时毛利高于动力电池5个百分点，是利润提升主驱动
   - **出海加速** (高潜力): 欧洲葡萄牙15GWh+东南亚泰国规划+北美Stellantis合作，海外占比2025年12% → 2028年30%+
   - **固态电池** (高估值): "无界"全固态电池2026年小批量装车，对标QuantumScape等技术领先公司，享受估值溢价
3. **业绩拐点确立:** 2024-2025年完成产能投入期，FCF转正；2025年净利率3.3% → 2027E预计6%+；ROE从3.5% → 8%+，业绩弹性显著

### 9.2 未来需跟踪的指标

- **国内动力电池市占率:** 月度装机量排名 (中创新航7-8%是否守住或上升)
- **储能业务收入及毛利率:** 季度披露，>20%毛利率为高景气
- **海外出货量占比:** 季度披露，>15%为加速
- **毛利率走势:** 季度披露，>18%为景气向上、<15%为价格战
- **应收账款周转天数:** 季报披露，<120天为健康
- **资本开支:** 半年报披露，FCF转正后资本开支是否回落
- **固态电池进展:** 公告披露，量产时间表
- **新客户拓展:** 海外车企定点公告 (Stellantis/大众等是否进入Tier 1)
