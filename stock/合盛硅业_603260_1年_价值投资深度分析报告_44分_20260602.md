## 一、投资结论

**评级:** 谨慎持有（周期底部博弈，非价值投资标的）
**当前股价:** ¥37.75
**当前市值:** ¥446.2亿
**加权合理估值:** ¥34.50 (对应市值 ¥407.8亿)
**安全边际:** -8.6% (不足——当前价格已反映部分复苏预期)
**预期年化回报:** -5%~+15%

核心观点:
1. 合盛硅业是全球工业硅（230万吨产能）和有机硅单体（173万吨）双料冠军，"煤-电-硅"一体化成本优势构筑了行业最深的成本护城河，在周期底部仍能维持正向经营现金流（2025年OpCF +40亿）
2. 2025年是公司上市以来首次年度巨亏（-29.91亿），主因光伏多晶硅资产减值（Q4单季减值约17亿）+工业硅/有机硅价格双杀。2026Q1已确认经营拐点：扭亏为盈（NP +0.77亿），毛利率回升至19.08%，有机硅DMC价格从底部11000元/吨反弹至15000元/吨（+36%）
3. 核心矛盾：有机硅已进入盈利修复周期（反内卷+海外产能退出），但工业硅仍处磨底阶段（产能过剩、全行业亏损），叠加公司65%高负债率和58亿定增压价，当前价格缺乏安全边际
4. 这是典型的周期股博弈，不符合巴菲特"以合理价格买入优质公司"的标准。更适合在有机硅持续涨价+工业硅供给侧改革信号出现后，以更低价位（¥28-32）建仓

## 五、行业与竞争分析

### 5.1 行业概况

- **行业名称**：硅基新材料（工业硅+有机硅）
- **行业规模（TAM）**：工业硅全球~650万吨/年（~800亿人民币）；有机硅全球~350万吨/年（~2000亿人民币）
- **行业增速**：工业硅近3年CAGR +8%（但2025年-5.6%）；有机硅近3年CAGR +6.5%
- **生命周期**：
  - 工业硅：**成熟期（周期化）**——渗透率>70%，增速放缓，价格围绕成本波动
  - 有机硅：**成熟期（第二增长曲线）**——传统应用（建筑密封胶）成熟，新兴应用（新能源车/5G/AI液冷/机器人）带来增量；供给端"反内卷"加速出清后进入盈利修复
- **行业集中度**：工业硅CR5 ~44%（合盛~27%第一）；有机硅CR5 ~62%（合盛~25%第一）
- **行业平均PE**：周期底部PE不可比；正常年份15-25x | **行业平均PB**：1.0-2.0x

### 5.3 护城河分析（industry-analyst + buffett框架交叉验证）

- **资源垄断（能源成本）**：新疆鄯善+石河子两大基地，自备热电厂+石英矿，工业硅完全成本<8,000元/吨（行业最低梯队），较云南丰水期成本低15%+，较西南枯水期成本低40%+。这是"地理/资源型成本优势"，属于buffett分类中最持久的壁垒类型之一
- **规模经济**：工业硅230万吨（全球第一）+有机硅173万吨（国内第一），单吨固定成本远低于同行。新进入者需投资200亿+、建设3-5年才能达到同等规模
- **一体化闭环**："煤→电→工业硅→有机硅→深加工"，降低中间环节利润漏出。尤其在电价上行周期，自备电厂是决定性成本优势
- **品牌/转换成本**：无——有机硅和工业硅为标准化产品，客户按纯价格决策
- **护城河宽度：宽（成本优势型+规模经济），但处于commodity行业，护城河仅在周期底部体现防御价值**

## 六、估值分析

### 6.1 PE估值

- 当前PE (TTM): 亏损无意义
- 2026E PE: 37.75 / (12.0/11.82) = 37.2x（基于共识净利润~12亿）
- 2027E PE: 37.75 / (22.0/11.82) = 20.3x（基于共识净利润~22亿）
- 5年历史PE: 最低~10x（2021年业绩巅峰期PE反而最低，周期股特征）/ 中位~20x / 最高~100x+
- 当前PE历史分位: 不适用（亏损）
- 行业PE: 周期底部不可比
- 合理PE判断: 取周期中位净利润15-20亿，给予15-18x（周期股折价）。合理PE基于2026E 12x → 需折让
- 使用mid-cycle EPS 1.50 (归一化利润~17.7亿)，合理PE 18x
- **合理估值: ¥27.00**（1.50 × 18），距当前 -28.5%

> PE估值在周期股中具有误导性——盈利最低时PE最高（甚至为负）。巴菲特的Graham传统做法是使用7-10年平均利润。过去7年（2020-2026E）均值约24.6亿（含超级周期年份），最近5年（2022-2026E）均值约17.2亿。取归一化利润约18亿（EPS 1.52），合理PE 15-20x。

### 6.2 PEG估值

- 预测3年净利润CAGR (2025→2028E): 从-29.91亿→~29.42亿（开源证券预测），扭亏为盈，CAGR无法计算
- 从2026E→2028E: (29.42/12.0)^(1/2)-1 = 56.6%
- PEG (2026E): 37.2 / 56.6 = 0.66（若增长兑现则低估）
- PEG=1对应合理PE: 57x...不切实际
- PEG估值不适用于强周期股，参考意义有限
- **PEG估值: 暂不纳入**

### 6.3 PB估值（周期股核心方法）

- 当前PB: 1.52x | 行业PB: 0.8-2.5x（周期底部1.0x以下，周期顶部2.5x+）
- BVPS: ¥24.88（2026Q1）
- 2026E ROE: ~4.5%（12 / ~267亿平均净资产，含定增稀释）
- 折现率（CAPM）: 10.0%（Rf 2.8% + β1.2 × ERP 6%）
- 合理PB = ROE / 折现率 = 4.5% / 10.0% = 0.45x（仅看当前ROE太低）
- 使用normalized ROE 7%（mid-cycle）= 7%/10% = 0.70x → ¥17.42
- 但市场通常给予成本优势龙头10-30% PB溢价。给予normalized PB 1.2-1.5x
- **合理估值（PB）: ¥30-37**（取1.2-1.5x BVPS 24.88）

> 周期股估值的核心逻辑：在ROE低谷（甚至为负）时PB仍高于1.0x，反映市场对公司未来盈利恢复的预期。合盛1.52x PB说明市场已定价了部分复苏。

### 6.4 DCF估值

- WACC: 9.5% | 永续增长率: 2%
- 基准FCF (2025): ~10亿（OpCF 40亿 - 维持性Capex ~15亿 - 扩张性Capex ~15亿）

## 九、核心逻辑与风险

### 9.1 核心增长逻辑

1. **有机硅反内卷供给出清**：行业自律限产+陶氏英国14.5万吨退出+2026年零新增产能，供需格局从过剩走向紧平衡。合盛173万吨产能的利润弹性极大——DMC每涨1000元/吨，年化增利约10-12亿
2. **工业硅成本优势护城河**：新疆煤电一体化成本<8,000元/吨（行业最低），在全行业亏损时仍可保本。若国家出台供给侧改革（淘汰12500KVA以下炉型，去化>35%产能），将触发趋势性价格反弹，合盛作为最低成本生产商获利最大
3. **碳化硅第二曲线**：6/8英寸衬底已量产（宁波+呼和浩特基地），12英寸研发中。碳化硅是新能源车/充电桩/AI电源的核心材料，市场空间大，若放量将提升公司估值中枢（从周期化工→成长材料）

### 9.2 未来需跟踪的指标

- **有机硅DMC周度报价**（卓创资讯/百川盈孚）：持续>14,000元/吨为健康，>16,000元/吨为高景气。跌破12,000需警惕反内卷格局松动
- **工业硅月度开工率**：若行业开工率从当前~49%降至<40%并持续2个月，说明产能出清加速
- **季度毛利率**：Q1为19%，若Q2/Q3维持>18%说明主业修复持续；若回落至<15%需重新评估
- **负债率和短期债务**：当前短期债务101.81亿，关注2026H2到期偿还情况和定增进度
- **光伏资产处置/止血进展**：若公司公告出售或永久关停光伏资产，是重大正面催化剂

**护城河定性复核（基于buffett 03-business-moat.md）:**

合盛硅业属于buffett框架中的**最低成本生产商**型护城河——"当一家公司销售具有商品特性的产品时，成为低成本生产商至关重要"（2000年致股东信）。

- 新疆煤电硅一体化构成"地理/资源型成本优势"，属于buffett分类中最持久的壁垒类型——"very high, non-replicable"
- 但buffett同时警告：commodity行业中的成本优势可能不够宽。南方纺织业的案例表明——当全球竞争使得整个行业的基本经济学变得太差时，局部优势无法抵御大环境
- 与GEICO类比：GEICO的低成本优势持续了50年+，但保险业有品牌/直销/数据网络效应等多重壁垒；合盛仅有成本+规模壁垒，更脆弱
- 护城河评级：**中等宽度（但处于commodity行业，防守性不如消费品/服务业中的同等宽度护城河）**
- 与Phase 5结论**一致**：宽护城河（成本优势型），但属于commodity行业

**估值纪律检查（基于buffett 06-valuation-capital.md）:**

- 安全边际是否充足（>20%）：**否**——当前PB 1.52x，高于周期底部合理PB（1.0-1.2x），安全边际为负
- 是否在能力圈内：**是**——硅基化工产业链简单清晰
- 是否受市场情绪影响：**是**——市场可能在押注有机硅涨价而没有充分考虑高负债风险

**buffett 综合定性意见:**

该标的**不符合**巴菲特核心投资标准。巴菲特会说："这是一家好行业中成本最低的生产商——但它所处的行业不是好行业（commodity），而且管理层用借来的钱做了愚蠢的扩张（光伏多晶硅）。在¥37这个价位，你在赌有机硅继续涨价、工业硅触底反弹、光伏止血——三个赌注必须都兑现才能赚到钱。我不做需要这么多事情都做对的投资。如果PB回到1.0x以下（¥25），至少安全边际有了，那时候才值得看一眼。"
