## 六、估值分析

### 6.1 PE估值
- 当前PE (TTM): 212.70x
- 5年历史PE: 最低~40x / 中位~80x / 最高~300x
- 当前PE历史分位: ~75%
- 行业PE: 台积电~20x / 联电~12x / 行业均值~15-20x
- 合理PE判断: 考虑到中芯国际处于产能扩张周期的利润低谷期，且具备稀缺性溢价，给予2026E PE 150-180x（基于2026E EPS 0.80-0.90元）
- **PE合理估值: ¥120-162**（基于2026E净利润55-65亿）

### 6.2 PEG估值
- 预测3年净利润CAGR: ~30%（2025年50亿→2028E约110亿，受益于产能爬坡+ASP提升+折旧高峰过去）
- PEG = 212.70 / 30 ≈ 7.09
- PEG >> 2 → 按传统PEG框架严重高估
- 但需要注意：当前净利润被折旧严重压低，若以折旧前利润（NP+折旧≈323亿）计算，PEG约1.0-1.5
- **PEG估值参考: 不适用于当前阶段，需以EV/EBITDA替代**

### 6.3 PB估值
- 当前PB: 7.11x | 行业PB: 台积电~7x / 联电~1.8x / 格芯~2.5x
- ROE: 3.40% | 折现率（COE）: ~8%
- 合理PB = ROE / COE = 3.40% / 8% = 0.43x（严重高估）
- 若以正常化ROE 8%估算（折旧高峰后）：合理PB = 8% / 8% = 1.0x
- 市场给予的PB溢价反映的是"稀缺性溢价+国家战略溢价+未来盈利改善预期"
- **PB估值参考: 当前PB严重脱离基本面，但反映了期权价值**

### 6.4 DCF估值
- WACC: 7.5% | 永续增长率: 3%
- 基准FCF: 当前FCF为-399亿，无法直接用于DCF
- 改用正常化盈利：假设5年后折旧高峰过去，FCF转正
- 简化两阶段DCF：
  - 阶段1（2026-2030）：OCF年均增长15%，Capex年均500亿，FCF仍为负
  - 阶段2（2030后）：FCF = 年均300亿，永续增长3%
  - Terminal Value (2030) = 300 / (7.5%-3%) = 6,667亿
  - 折现到2026年 = 6,667 / 1.075^5 ≈ 4,600亿
- 当前市值约10,737亿，远超DCF推算，说明市场定价了远超基本面的"战略期权价值"
- **DCF估值: 无法在合理假设下达到当前市值，传统DCF方法对此类战略资产失效**

### 6.5 EV/EBITDA估值（替代方法）
- 2025年EBITDA ≈ NP 50.41 + D&A 273.07 + 利息支出(估)10 + 所得税(估)10 ≈ 343亿
- 企业价值(EV) ≈ 市值10,737 + 净有息负债(估)500 ≈ 11,237亿
- EV/EBITDA ≈ 32.7x
- 台积电EV/EBITDA ≈ 12-15x / 联电 ≈ 6-8x
- 若给予中芯国际EV/EBITDA 25x（考虑稀缺溢价+增速溢价）：合理EV ≈ 8,575亿，合理股价 ≈ ¥100
- 若给予EV/EBITDA 30x：合理EV ≈ 10,290亿，合理股价 ≈ ¥122
- 若给予EV/EBITDA 35x（乐观情景）：合理EV ≈ 12,005亿，合理股价 ≈ ¥143

## 九、核心逻辑与风险

### 9.1 核心增长逻辑
1. **AI需求外溢（行业层面）**: 全球AI芯片需求爆炸→台积电/三星先进制程产能被AI巨头包揽→电源管理/模拟/网通等配套芯片需求外溢至成熟制程代工厂→中芯国际ASP和产能利用率双升
2. **国产替代不可逆（政策层面）**: 美国技术封锁持续→中国半导体自主可控需求迫切→中芯国际作为唯一先进制程国产平台，享受确定性的需求增量
3. **折旧高峰后的利润弹性（公司层面）**: 2025-2027年是资本开支高峰对应的折旧高峰，此后折旧增速放缓→收入继续增长→利润剪刀差释放，ROE有望从3.4%向8-10%回归

### 9.2 未来需跟踪的指标
- **季度ASP趋势**: 晶圆平均售价是否持续上行，是验证AI外溢效应的关键指标；关注每片晶圆收入变化
- **产能利用率**: 维持>90%是健康信号；若跌破85%需警惕需求回落
- **折旧/收入比**: 当前约40%，若该比率持续下降说明收入增长正在消化折旧压力
- **Q2 2026实际vs指引**: Q2指引环比+14-16%为全球最高，实际能否兑现是关键验证点
- **美国对华半导体政策变化**: 任何新的出口管制/实体清单升级将直接冲击估值逻辑

|---|---------|------|------|------|----------|

| 3 | 折旧持续高企压制利润，ROE长期无法改善 | 公司 | 高 | 中 | 折旧高峰预计2027年后缓解；关注OCF而非NP |

### 9.4 关键假设与证伪条件
- **核心假设**: AI需求外溢+国产替代将推动中芯国际2026-2028年营收CAGR >20%，并在折旧高峰后实现利润弹性释放
- **证伪条件**: 若2026H2营收增速回落至<10%且产能利用率降至<85%，说明AI外溢逻辑不成立，需重新评估投资逻辑
