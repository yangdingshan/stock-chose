## 五、行业与竞争分析

### 5.1 行业概况

- **行业名称**: 白酒(申万二级:饮料制造 → 申万三级:白酒)
- **行业规模**: 2024年白酒规模以上企业销售收入约7,500亿,同比+2.1%
- **行业增速**: 2024年+2.1% (中国酒业协会《白酒消费白皮书2024-2025》),进入低速期
- **生命周期**: **成熟期(周期化)** — 渗透率视角:白酒消费已从政策消费转向商务/礼品消费,渗透率见顶,但高端白酒受益于消费升级,呈现"成熟期+第二增长曲线"特征
- **行业集中度**: CR3约45%(茅五泸),CR5约62%(茅五泸汾洋),行业出清后龙头集中度持续提升

### 5.3 护城河分析 (按Buffett框架)

- **品牌/稀缺资源**: 极强 — "五粮液"为中国八大名酒之一,千年老窖、明初古窖池群(始于1368年)为不可复制的稀缺资源,品牌力仅次于茅台
- **转换成本**: 高 — 浓香型白酒消费者口味具有强粘性,渠道经销商经过长期合作形成网络效应
- **网络效应**: 中 — 宴请/礼品场景下品牌越知名,被选择概率越高,正反馈循环
- **规模经济**: 强 — 38万吨储能、年产约15万吨的优质基酒储备,行业第二大产能
- **成本优势**: 中 — 高端白酒毛利率均在75%以上,成本非核心竞争要素
- **定价权**: 强 — 2024年八代普五出厂价从969→1019元提价50元(5.2%),批价稳定在970-1019元区间;1618定位900-1000元腰部价格带,与飞天茅台的500-1500元价差形成补位

**护城河宽度评估: 宽(强品牌 + 稀缺资源 + 规模 + 定价权四重叠加)**

### 5.4 行业政策与PEST

- **P(政策)**: 2026-06-01起施行《白酒消费税计算明细表》新规,要求"出厂价与关联方对外售价60%孰高"计税,直接利好规范经营的头部酒企,中小酒企/贴牌商出清加速;2025-2026年行业"反内卷"政策导向延续
- **E(经济)**: 中国GDP增速5%左右,M2持续宽松,商务消费弱复苏,礼品消费承压
- **S(社会)**: 人口老龄化 + 年轻人白酒消费意愿下降,长期需求承压;但高端白酒的"少喝酒、喝好酒"消费升级逻辑成立
- **T(技术)**: 无颠覆性技术替代风险(白酒为传统消费品);数字化营销和私域运营是行业新趋势

- **合理估值: ¥90.00** (按2025E EPS 6.95 × 13X = ¥90;按2026E EPS 7.20 × 12.5X = ¥90)

### 6.2 PEG估值

- 预测3年(2025-2027)净利润CAGR: +3% (假设2025出清,2026-2027恢复5%增长)
- PEG = 静PE 35.7 / CAGR 3 = **PEG > 11** (典型价值陷阱特征,需用动态PE重估)
- 动态PE (2026E) = 当前价 ¥81 / 2026E EPS ¥7.20 = 11.3X
- PEG (动态) = 11.3 / 3 = 3.8 (依然偏贵)
- **结论**: PEG估值模型在出清年份失真,需以2026-2027业绩复苏后的PEG=2合理推算
- **PEG=1对应估值: 2026E EPS ¥7.20 × 合理PE 12.5X = ¥90.00**

### 6.3 PB估值

- 5年历史PB: 最低2.0 / 中位4.5 / 最高8.5

- 行业PB: 4.71 (均值)
- ROE: 24.6% / 折现率: 9% (WACC估算) → 合理PB = ROE/折现率 = 24.6/9 = 2.73X

## 九、核心逻辑与风险

### 9.1 核心增长逻辑

1. **行业出清末期**: 2024-2025年完成深度出清(库存/批价/经销商三方承压),2026年进入修复通道
2. **品牌护城河深厚**: 八代普五+1618+39度形成"高端三层"产品矩阵,五粮液品牌力位居行业第二
3. **改革红利释放**: 营销2.0版本、专营公司设立(五钧/五浚)、八代合同量减少30%,渠道精细化
4. **消费税新规利好龙头**: 2026-06-01新规实施,中小酒企/贴牌商出清加速,头部酒企集中度提升
5. **高股息防御性**: 2024年度分红率65%,股息率4.45%,在低利率环境下提供安全垫
6. **估值已部分修复**: PE 25X处于历史中位,PB 2.5X处于历史低位(~15%分位)

### 9.2 未来需跟踪的指标

- **八代普五批价**: 站稳970元是核心信号,跌破950元是预警 (监测: 今日酒价、批价内参)
- **渠道库存**: 健康水平1-2个月,超过3个月为预警 (监测: 经销商调研)
- **2026Q2-Q4季度业绩**: 验证2026Q1开门红是否延续 (监测: 中报/三季报)
- **系列酒增速**: 反映腰部价格带补位成效 (监测: 2026年中报分产品收入)
- **经销商打款积极性**: 反映渠道对未来信心 (监测: 2026新财年打款进度)
