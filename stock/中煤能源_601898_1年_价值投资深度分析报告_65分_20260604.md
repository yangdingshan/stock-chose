# 中煤能源 (601898) 价值投资深度分析报告

分析日期: 2026-06-04 | 分析周期: 1年

---

## 一、投资结论

**评级:** **持有 (中性偏谨慎)**
**当前股价:** ¥17.20
**当前市值:** ¥2,281亿 (132.59亿股 × ¥17.20)
**加权合理估值:** ¥15-18 (对应市值 ¥1,990-2,387亿)
**安全边际:** +0% ~ -15% (基本合理,略有高估)
**预期年化回报:** -2% ~ +6% (含股息)

核心观点 (3-5句总结):
1. **资源最多+估值最低是核心标签**:中煤能源煤炭**可采储量138.7亿吨(行业第一)、可采年限103年**(神华注入后67年、陕煤约30-50年),**PE 12.88/PB 1.39是煤炭三大龙头中最低**,具备"价值陷阱"的初步条件之一(估值低)。
2. **但派息率持续下降是核心警示**:2024年35% → 2025年28.37% → 2025年中期分红仅每股0.217元(末期);**派息率连续下降削弱了"红利股"的核心卖点**,这是中煤能源与神华(75%分红率)和陕煤(60%分红率)最大的差距。
3. **煤化工业务是"双刃剑"**:煤化工收入占比12.51%(远高于神华1.72%、陕煤0%), 中天合创等大型项目;**2025年油价低迷拖累煤化工业绩**(聚乙烯/聚烯烃价格跌),但2026年中东地缘紧张推升油价+聚乙烯价格已涨10% → 2026年煤化工是**业绩弹性来源**。
4. **ROE偏低(10-13%)+ 杠杆较高(45%)是估值压制**:中煤ROE 10-13%(神华12-14%、陕煤19.79%), 资产负债率45.02%(2025H1, 神华25%、陕煤43.7%);**低ROE+高杠杆的组合,既压制估值,又增加周期下行风险**。
5. **关键判断:这是一只"低估值+低股息+高弹性"的煤炭股**,适合作为**周期股组合配置**(仓位比神华/陕煤小),**最大风险是派息率持续下降**;建议**持有10-20%仓位作为低估值底仓**, 关注派息率能否企稳回升至30%以上。

- **集团资产整合预期**:中煤集团旗下仍有未上市资产,长期整合空间存在(类似神华2026年完成的资产注入)

- **行业规模**: 全国煤炭年产量约47亿吨,中煤产量占全国2.9%(自产煤)
- **行业增速**: -1% ~ 0% (2024-2025年);中长期受双碳政策压制
- **生命周期**: **成熟期+周期化**
- **行业集中度**: CR8约35%,中煤是行业第三
- **行业平均PE**: 22.04 (TTM);行业中值PE 23.56

### 5.3 护城河分析 (Buffett视角)

| 护城河类型 | 评估 | 说明 |
|----------|------|------|
| **资源垄断** | 强 | 保有储量266.5亿吨,可采138.7亿吨,**行业第一**;可采年限103年**行业最长** |
| **规模经济** | 中 | 1.38亿吨产量(行业第3),略低于神华(3.27)和陕煤(1.75) |
| **成本优势** | 中 | 大海则等新矿产能释放;2025年毛利率提升+2.61pct |
| **煤化工一体化** | 中强 | **中天合创等大型煤化工项目,行业独有差异化** |
| **一体化(纵向)** | 弱 | **无铁路+港口+航运体系** (神华独有) |
| **转换成本** | 弱 | 客户主要为电厂/化工,转换成本低 |
| **品牌/特许** | 弱 | 动力煤+煤化工产品同质化严重 |
| **分红记录** | 弱 | 派息率2024 35% → 2025 28.37%,**连续下降** |
| **去杠杆** | 中 | 资产负债率持续下降(47.69% → 45.02%) |

**护城河宽度**:**中** (资源强+煤化工差异化+无一体化+ROE偏低+派息率下降, 整体比神华**窄一档**, 接近陕煤)

**Buffett判断**:
- 资源储量优秀(可采年限103年) + 煤化工差异化是相对优势
- 弱点: ROE偏低 + 高杠杆 + **派息率持续下降** + 无一体化
- 派息率下降是"机构本能"风险信号 — 央国企可能为"扩张"或"保安全"而压低分红
- 这是"**有安全边际但缺乏强护城河**"的典型"价值股"画像

### 6.2 PEG 估值

- **2026E EPS 1.56 vs 2025E 1.35**: 同比+15.41%(从业绩底部恢复)
- **3年CAGR (2025-2028)**: 假设煤价企稳+煤化工恢复, 净利润从179→206→220→235亿, CAGR约+9.5%
- **PEG = PE / CAGR = 12.88 / 9.5 = 1.36** (合理)
- PEG=1对应PE 9.5x → 合理估值: 1.56 × 9.5 = **¥14.82**
- 周期股PEG失真,需结合PB/平均盈利
