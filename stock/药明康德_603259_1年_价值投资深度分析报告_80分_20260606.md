## 六、估值分析

### 6.1 PE估值

- **当前PE (TTM)**: 15.06倍 (基于2025归母净利191.51亿)
- **当前扣非PE (TTM)**: 19.96倍 (基于2025扣非净利132.41亿, 更能反映真实经营)
- **5年历史PE区间**: 15-60倍, 中位~28倍, 均值~30倍

- **行业平均PE**: CXO行业A股25-40倍, 中位~30倍

- **合理估值**: 5.91元 × 22-25倍 = **¥130-148**, 中位 **¥138**
- **距当前**: ¥138/96.56 - 1 = **+42.9%** (充足)

### 6.2 PEG估值

- **预测3年 (2026-2028) 净利润CAGR**: 基于2025经调整non-IFRS 149.57亿, 2028E估计230亿, CAGR **+15.4%** (扣非口径)
- **PEG = 扣非PE / CAGR = 19.96 / 15.4 = 1.30** (PEG>1, 估值偏贵但成长性较强)
- **按PEG=1**: 合理PE = 15.4倍 → 合理估值 = 5.91 × 15.4 = ¥91
- **按PEG=1.2** (考虑行业龙头+订单可见度): 合理PE = 18.5倍 → 合理估值 = 5.91 × 18.5 = **¥109**
- **结论**: PEG法相对保守, 给出估值下限 **¥109**

### 6.3 PB估值

- **当前PB**: 3.42倍 (96.56 / 28.23)
- **5年历史PB区间**: 2.5-9倍, 中位~4.5倍
- **行业平均PB**: CXO行业A股2-4倍, 中位~3倍
- **ROE (2025A)**: 24.3%
- **合理PB = ROE / 折现率 = 24.3% / 9% = 2.70** (经典Gordon模型)
- **考虑成长性调整**: 合理PB = 3.5-4.5倍
- **合理估值**: 28.23 × 3.5-4.5 = **¥99-127**, 中位 **¥113**

### 6.4 DCF估值

- **核心假设**:
  - WACC = 9.0% (Rf 2.5% + Beta 1.0 × ERP 6.5%)
  - 永续增长率 g = 3.0%
  - 2026-2028 FCF增速 15% / 13% / 10%
  - 2026E经调整non-IFRS归母净利 180亿
- **FCF预测** (基于经调整non-IFRS净利):
  - 2026E: 180亿
  - 2027E: 207亿
  - 2028E: 228亿
  - 终值 = 228 × 1.03 / (0.09 - 0.03) = 3,914亿
- **DCF合理估值**:
  - PV(2026-2028) = 180/1.09 + 207/1.09^2 + 228/1.09^3 = 165.1 + 174.3 + 176.1 = 515.5亿
  - PV(终值) = 3,914/1.09^3 = 3,024亿
  - 合计企业价值 = 3,540亿
  - 加净现金 200亿 (现金及金融资产-有息负债)
  - 股权价值 = 3,740亿
  - **每股价值 = 3,740 / 29.84 = ¥125**
- **敏感性**: ±20% FCF → ¥100-¥150
- **DCF合理估值**: **¥125**

## 九、核心逻辑与风险

### 9.1 核心增长逻辑

1. **CRDMO一体化龙头"卖水人"模式**: 全球新药研发支出持续增长 (2026年突破3000亿美元), 药明康德作为全球Top 1 CRDMO平台, 客户从临床前到商业化全程委托, 收入可见度强, 受益全球生物医药投融资回暖。
2. **TIDES (多肽/寡核苷酸) 第二增长曲线**: 受益全球GLP-1减重药爆发 (司美格鲁肽/替尔泊肽) + 专利到期后仿制潮 + ADC新分子需求, 2024年+70%、2025H1+141.6%, 产能10万升+为全球第一, 长期高增长可期。
3. **订单+业绩+估值三重共振**: 在手订单597.7亿(+23.6%)创历史新高, 2026Q1扣非+83.56%验证业绩拐点, 当前PE 15倍处历史10%分位, "业绩+订单+估值"3 in 1 重估机会。

### 9.2 未来需跟踪的指标

- **季度在手订单增速**: 持续>20%表明景气持续, 跌破10%需警惕
- **TIDES业务季度收入**: 持续>30%同比增速, 跌破20%表明赛道见顶
- **毛利率**: 维持50%+为健康, 跌破45%需关注成本/价格压力
- **美国客户收入占比**: 维持55-65%健康, 跌破50%或激增>70%都是地缘风险信号
- **《生物安全法》立法进展**: 美国国会每半年审议, 列入NDAA或单独立法均为重大利空
- **新建产能爬坡**: 常州/泰兴/金山/新加坡/美国基地的产能利用率
- **重大并购/股权处置**: 类似2025年合联生物处置, 需关注主业聚焦度
