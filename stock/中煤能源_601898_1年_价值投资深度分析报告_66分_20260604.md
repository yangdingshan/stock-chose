## 五、行业分析：煤炭行业2026年

### 5.1 产业生命周期定位

**煤炭行业处于：成熟期(周期上行阶段)**

- 中国煤炭消费总量已进入平台期(40-45亿吨/年)
- 渗透率已过峰值(煤炭在一次能源占比从68%降至55%左右)
- 但新能源替代是20-30年级别的渐进过程，煤炭在未来10-15年仍是能源基石
- **2026年特征**：成熟期中的周期性上行——供给收缩+需求意外增长

### 5.2 2026年煤炭行业供需格局

**供给端 — 内外双紧**：
- 山西矿难后77座煤矿仍停产(涉及产能7,605万吨)，复产进度缓慢
- 印尼出口配额收紧，全年出口预计减量6,000-7,000万吨
- 澳大利亚、俄罗斯进口同步下滑
- 国内"反内卷"政策加速中小煤矿退出

**需求端 — 多重增量**：
- 南方用电负荷提前45天创新高，夏季用电高峰将至
- 厄尔尼诺气候预警，2026年夏秋季可能出现极端高温
- AI算力集群带动数据中心配套火电增长
- 中东冲突推高油气价格 → 煤化工对煤炭原料需求增加
- 2026年全社会用电量预计增长5.5%

**煤价走势**：
- 秦皇岛港5500大卡动力煤：当前860元/吨(较2025年底678元涨27%)
- 全年煤价中枢预计：800-850元/吨(显著高于2025年的700元)
- Q3旺季高点可能触及900-1,000元/吨
- 焦炭已连续五轮提涨

### 5.3 波特五力分析

| 力量 | 强度 | 分析 |
|------|------|------|
| **现有竞争者** | 中等 | 行业集中度提升(CR5约45%)，神华为绝对龙头；中煤为第二大 |
| **新进入者** | 弱 | 矿业权获取极难+"双碳"政策限制新批矿权 |
| **替代品威胁** | 中长期增强 | 风光新能源替代是20-30年级别趋势，但短期不构成威胁 |
| **供应商议价力** | 弱 | 央企自有矿山，上游从属于自身 |
| **客户议价力** | 中等 | 电力企业(五大发电集团)集中度高，但年度长协锁定大部分价格 |

### 5.4 煤炭行业估值逻辑

- 煤炭是典型的**周期股**：PE最低时往往是周期顶点，PE最高时反而可能是底部
- 当前处于**盈利下行→企稳回升**的拐点阶段
- 适用估值方法：历史PB band + 正常化盈利PE + DDM(分红贴现)
- 行业合理PE中枢：8-14x，取决于煤价预期方向

### 6.3 DDM (股利贴现) 估值

```
假设：
- 2026年分红: 0.25元/股 (基于30%分红率+1.60 EPS)
- 长期股息增长: 3%
- WACC: 8%
内在价值 = 0.25 / (0.08 - 0.03) = ¥5.00 → 偏低(DDM不适用增长期公司)
```

> DDM估值偏低是因为公司分红率仅28-30%，大部分利润留存再投资。参考华泰证券采用4% WACC + 35.6%分红率得出目标价20.81元。
